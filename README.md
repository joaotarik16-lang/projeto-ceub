Análise de Emendas Parlamentares com Pandas
O projeto contém uma série de exercícios utilizando a biblioteca **pandas** em Python para análise de dados de emendas parlamentares.

O objetivo é praticar:
- Limpeza de dados
- Filtragem
- Transformação
- Agrupamento

Os dados utilizados estão no arquivo `emendas_tratamento.parquet`.

---

Estrutura do Projeto:

- `pandas_intro_ex2.ipynb` → Notebook com os exercícios
- Dataset utilizado:
  https://drive.google.com/file/d/19ydA23-eboyiKxN29JL1EQWW4CplTiKg/view

Tecnologias Utilizadas:

- Python
- Pandas
- Jupyter Notebook
